# MoviePilot 专属增强插件库 (MoviePilot Plugins Suite)

本仓库是为 MoviePilot (V2+) 量身定制的第三方专属增强插件库，包含一系列极具实用价值的高级系统级及媒体管理类插件。

---

## 📦 插件列表与核心功能

### 1. 🛠️ 环境变量编辑器 (`enveditor`)
> 告别频繁重启容器！实现在线实时编辑、热重载与增量更新系统配置及环境变量的利器。

* **📂 在线文件管理**：直接在线读取、编辑与保存宿主主机的 `app.env` 物理配置文件（默认路径 `/config/app.env`）。
* **⚡ 增量热更新**：保存配置后自动执行增量更新算法，瞬间同步至当前运行中的 Python 进程内存以及核心系统配置对象 `settings`，无需重启 Docker 容器即可生效！
* **📜 智能格式化排版**：支持一键开启“模板化格式化与排版”，自动将零散的环境变量按照系统大类进行对齐、整理并补全中文注释（如：系统配置、基础配置、用户界面、媒体解析、媒体下载、通知渠道、插件配置等）。
* **⏰ 历史变更追溯**：内置最大保存 20 次的历史备份机制，并在 UI 中直观地展示每一次保存的变更细节（包括新增、修改、删除的具体字段及旧值对比），安全无忧。
* **🧹 极简轻量**：极致优化，保存时剔除 SQLite 数据库中的冗余大数据缓存，确保数据唯一物理实体存放于 `app.env`，防范引号变异或配置冲突。

---

### 2. 🎬 本地轻量字幕下载器 (`localsubdownloader`)
> 专为 PT 玩家/高清影音爱好者打造的本地字幕全自动与手动匹配下载引擎，带你实现“字幕自由”。

* **🤖 双模运作**：支持整理入库时**自动匹配下载**，同时提供高颜值的**控制台手动交互整理界面**。
* **🔌 多源融合检索**：原生接入 **Shooter (射手网)**、**Xunlei (迅雷字幕)**（以上两款无需 Token 即可实现精准对齐匹配），以及 **ASSRT (伪射手网)** 和 **A4k/SubDL** 接口（输入个人 Token 即可畅快下载）。
* **🎯 精准对齐与去重**：引入先进的 **MD5 内容去重** 与 **时间轴特征对齐算法**，智能排除低质及重复字幕。
* **💬 混合字幕检测**：集成智能 `ffprobe` 物理文件分析，实时解析并带缓存地读取视频中已存在的外置/内置/内挂字幕轨道，以精简的 `中文/英文/其它` 指标呈现在界面中，避免重复下载。
* **📂 面板深度集成**：控制台支持自定义路径切换、排序切换（按名称/时间）、关键字搜索过滤以及一键进入下级子目录，配合批量一键勾选，极速整理整季美剧/动漫字幕。

## 🚀 安装与部署

在 MoviePilot 插件市场中添加本仓库为第三方插件源即可轻松安装：

1. 打开 **MoviePilot 网页控制台**。
2. 进入 **系统设置** -> **插件市场**。
3. 在第三方插件源或仓库地址中填写本仓库的 GitHub 地址：
   ```text
   https://github.com/318182456/MoviePilot-Plugins
   ```
4. 点击更新或同步，即可在插件市场列表中看到 **环境变量编辑器** 和 **本地轻量字幕下载器**，一键点击安装即可完成部署。

## 🛠️ 技术开发规范

如果您需要继续扩展或贡献此项目，请遵循以下开发约定：

* **时区与编码**：一律采用 `UTF-8` 编码，业务时区统一使用 `JST/CST` 处理。
* **版本与元数据**：每次发布或修改插件代码，请务必同步更新插件目录主类 `__init__.py` 中的 `plugin_version`，以及根目录 `package.v2.json` 中对应的插件版本信息。
* **安全声明**：请勿在代码或配置文件中硬编码任何敏感凭证（如 Token、密码、Cookie 等）。
